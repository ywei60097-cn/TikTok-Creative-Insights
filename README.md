# TikTok Creative Insights

Creative Insights skill and report workspace for tracking popular Douyin/TikTok China AI ecommerce video references.

## Contents

- `skills/creative-insights/`: Codex skill for searching, validating, scoring, and summarizing AI ecommerce video examples by category.
- `reports/`: generated Creative Insights reports and smoke tests.

## Default Workflow

Use the `creative-insights` skill to:

1. Search recent Douyin/TikTok China AI ecommerce video examples.
2. Group credible links by category, such as appliances, photography, drones, food, apparel, beauty, home, pets, and digital products.
3. Summarize creative patterns, evidence quality, and concrete remake directions for AI ecommerce video generation.

The skill keeps strict source rules: do not invent links or metrics, label reposts as secondary sources, and mark unverifiable engagement as unverified.
