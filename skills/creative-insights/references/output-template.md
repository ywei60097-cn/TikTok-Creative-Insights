# Output Template

Use Chinese unless the user requests another language.

## Daily Brief Shape

Title:

`AI电商视频 Creative Insights｜YYYY-MM-DD`

Opening:

- `范围`: platform, time window, categories.
- `筛选标准`: briefly state that only attributable links are included.
- `结论`: 2-3 high-level trend observations.

Category section:

```markdown
### 品类名

1. **视频/案例标题或可识别描述**
   - 链接: [来源名](URL)
   - 来源: 原始/二手来源；账号/平台 if visible
   - 热度: visible metrics, ranking context, or `互动数据未公开/未核验`
   - 创意模式: pattern tags
   - 可借鉴点: one sentence on hook, visuals, product proof, or sales logic
   - AI复刻方向: one concrete remake idea for AI ecommerce video generation
```

Cross-category section:

```markdown
## 今天值得复用的创意结构

- Pattern: how it appears and why it works.
- Pattern: how to adapt it across categories.
```

Production recommendations:

```markdown
## 给AI电商视频生成的建议

1. Recommendation tied to script/hook.
2. Recommendation tied to visuals/assets.
3. Recommendation tied to editing/testing.
```

Data gaps:

```markdown
## 数据不足/待追踪

- Category: what was searched and why it was not included.
```

## Tone

Be practical, specific, and skeptical. Avoid vague phrases such as `很有爆款潜力` unless the brief explains the mechanism.
